# Privacy Policy

Last updated: 2025-12-03

初版公開

## 1. Overview

拡張機能『YouTube字幕抽出くん with AI整形』（以下「本拡張機能」）は、YouTube の動画から字幕データを取得し、ユーザーの操作に基づいて整形・保存する機能を提供します。

本拡張機能は、ユーザーのプライバシーを尊重し、個人情報を収集・共有せず、外部サーバーへのデータ送信も行いません。

## 2. Information We Collect

本拡張機能が取得するデータは以下のみです：

### 取得するデータ

- YouTube 動画から取得される 字幕データ（テキスト）

※ これはユーザーが開いているページの公開情報であり、個人情報には該当しません。

### 取得しないデータ

- 氏名、メールアドレスなどの個人情報
- YouTubeアカウント情報
- 閲覧履歴、Cookie、その他追跡情報
- その他、取得するデータに明記されていない一切の情報

## 3. Data Storage

取得した字幕データはユーザーのローカルストレージにのみ保存されます。
外部サーバーや開発者へ送信されることはありません。
ローカルストレージのデータはユーザー自身で削除できます。

## 4. Data Sharing with Third Parties

本拡張機能は、いかなるデータも第三者（外部サーバー / 開発者 / 広告会社など）への送信または共有を行いません。

## 5. Permissions Used

本拡張機能は動作のために以下の権限を利用します：

### activeTab

→ 現在表示中の YouTube ページの字幕データへアクセスするため

### storage

→ ユーザー自身のローカルストレージにデータを保存するため

### scripting / content scripts

→ YouTube ページ内から字幕を抽出するため

これらの権限は字幕抽出機能の提供に必要な最低限のものです。


## 6. Disclaimer（免責事項）

本拡張機能の利用により生じたいかなる損害・不利益についても、開発者は一切の責任を負いません。

- YouTube側の仕様変更
- ブラウザの動作
- データ消失
- 動作不具合
- 拡張機能の更新停止

などに起因する問題について保証いたしません。

また、予告なく本拡張機能の公開を停止する場合がございます。

本ポリシーは必要に応じて更新される場合があります。
変更点は本プライバシーポリシーの更新履歴にて告知いたします。
